    📚 Roteiro de Aprendizado

    Objetivo: Ensinar o ciclo de vida das requisições HTTP e a persistência de dados.

    • Passo 1: Selecione uma tabela → clique POST → preencha os dados → grave no banco.
    • Passo 2: Clique GET → veja o registro mais recente no terminal JSON.
    • Passo 3: Clique PUT → modal abre pré-preenchido com os valores atuais → edite → confirme.
    • Passo 4: Clique DELETE → registro removido do banco.
